# Python Quiz Game

## 프로젝트 개요
이 프로젝트는 콘솔 환경에서 동작하는 파이썬 퀴즈 게임이다.
사용자는 메뉴를 통해 퀴즈를 풀고, 새 퀴즈를 추가하고, 퀴즈 목록과 최고 점수를 확인할 수 있다.
퀴즈 데이터는 JSON 파일로 저장되며, 프로그램 재실행 후에도 상태가 유지된다.

## 퀴즈 주제와 선정 이유
- 주제: 대중적으로 많이 알려진 일반 상식/생활 밀착형 문제
- 선정 이유:
	- 많은 사용자가 별도 사전지식 없이 바로 참여 가능
	- 학습용 데모에서 문제 이해 시간이 짧아 기능 확인이 쉬움
	- 오답 선택지는 일부러 엉뚱한 답으로 구성하여 정답 구분이 직관적이고 플레이가 가볍게 느껴지도록 설계

## 실행 방법
1. 터미널에서 프로젝트 루트로 이동한다.
2. 아래 명령으로 프로그램을 실행한다.

```bash
python3 main.py
```

## 기능 목록
- 퀴즈 풀기
	- 문제를 순서대로 표시
	- 문제별 정답/오답 표시
	- 최종 점수와 결과 요약 출력
- 퀴즈 추가
	- 질문/보기(4개)/정답 번호 입력
	- 입력값 검증 후 `state.json`에 저장
- 퀴즈 목록 확인
	- 등록된 퀴즈 질문 목록 출력
- 점수 확인
	- 아직 응시하지 않은 경우 별도 안내 문구 출력
	- 응시 이력이 있으면 최고 점수 출력
- 저장/불러오기
	- 프로그램 상태를 JSON 파일로 저장
	- 상태 파일 손상 또는 누락 시 기본 데이터로 복구

## 파일 구조
핵심 파일 기준 구조는 다음과 같다.

```text
python-quiz-game/
├─ main.py
├─ quiz.py
├─ quiz_game.py
├─ input_handler.py
├─ quiz_bonus.py
├─ state.json
├─ state_default.json
├─ README.md
└─ .github/
	 └─ copilot-instructions.md
```

## 데이터 파일 설명 (state.json 경로/역할/스키마)
- 경로: `./state.json`
- 역할:
	- 현재 퀴즈 목록 저장
	- 최고 점수 저장
	- 퀴즈 응시 여부 저장
- 스키마:

```json
{
	"quizzes": [
		{
			"question": "string",
			"choices": ["string", "string", "string", "string"],
			"answer": "1~4 문자열"
		}
	],
	"best_score": 0,
	"has_played": false,
	"history": []
}
```
